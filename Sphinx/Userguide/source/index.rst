

MPCD Userguide
================================

This is the userguide for the *Multi-Particle Collision Dynamics* (**MPCD**) code developed by the `Shendruk Lab <https://tnshendruk.com>`_. 
It includes installation guidance, tutorials and explanations of the algorithm, and is primarily intended for people who wish to use-but-not-modify the code.
The code is available from `GitHub <https://github.com/Shendruk-Lab/MPCD>`_ and along with the `developer guide <https://github.com/Shendruk-Lab/MPCD/blob/master/docs/DevGuide.md>`_.
